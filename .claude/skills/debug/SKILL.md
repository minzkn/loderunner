# Debug Helper

게임 버그를 디버깅합니다.

## 사용법

버그 현상을 설명하면 관련 코드를 분석하고 원인을 찾습니다.

## 일반적인 버그 유형

### 1. 이동 관련
- 캐릭터가 벽을 통과함
- 캐릭터가 공중에 떠있음
- 사다리에서 움직이지 않음
- 낙하가 멈추지 않음

### 2. 적 AI 관련
- 적이 움직이지 않음
- 적이 벽을 통과함
- 적이 구멍에서 탈출 못함
- 적이 플레이어를 추적하지 않음

### 3. 충돌 관련
- 충돌이 감지되지 않음
- 잘못된 충돌 감지
- 함정이 작동하지 않음

### 4. 게임 상태 관련
- 스턱 감지 오작동
- 레벨 클리어 안됨
- 점수 계산 오류

## 디버깅 프로세스

1. **현상 분석**: 버그 재현 조건 파악
2. **코드 추적**: 관련 함수/메서드 찾기
3. **로직 검증**: 조건문, 계산식 확인
4. **원인 특정**: 버그 발생 지점 확인
5. **수정 제안**: 구체적인 코드 수정안 제시

## 출력 형식

```markdown
## 버그 분석
[현상 설명]

## 관련 코드
[코드 위치 및 내용]

## 원인
[버그 발생 원인]

## 수정 방안
[구체적인 코드 수정]
```
