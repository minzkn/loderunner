# Code Review

game.js 코드를 리뷰하고 개선점을 제안합니다.

## 리뷰 기준

### 1. 코드 품질
- 가독성
- 일관성
- 중복 코드
- 함수/메서드 크기

### 2. 버그 가능성
- 경계 조건 처리
- null/undefined 체크
- 타입 안정성
- 에러 처리

### 3. 게임 로직
- 물리 엔진 정확성
- 충돌 감지 정확성
- AI 행동 예측 가능성
- 상태 관리

### 4. 원작 충실도
- 1983년 Lode Runner 메카닉
- 플레이어 움직임
- 적 행동 패턴
- 레벨 디자인 원칙

### 5. 확장성
- 새 레벨 추가 용이성
- 새 기능 추가 가능성
- 설정 변경 용이성

## 출력 형식

```markdown
## 요약
[전체 코드 품질 요약]

## 잘된 점
- [장점 1]
- [장점 2]

## 개선 필요
### 심각도 높음
- [파일:라인] [문제 설명]

### 심각도 중간
- [파일:라인] [문제 설명]

### 심각도 낮음
- [파일:라인] [문제 설명]

## 리팩토링 제안
[구체적인 개선 코드 예시]
```
