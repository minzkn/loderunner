# Lode Runner Level Generator

새로운 게임 레벨을 생성합니다.

## 레벨 형식

- 그리드 크기: 28x16 (가로 x 세로)
- 마지막 줄(16번째)은 항상 `@@@@@@@@@@@@@@@@@@@@@@@@@@@@` (단단한 바닥)

## 타일 문자

| 문자 | 타일 | 설명 |
|------|------|------|
| `.` | 빈 공간 | 플레이어와 적이 통과 가능 |
| `#` | 벽돌 | 파괴 가능, 위에 서기 가능 |
| `@` | 단단한 벽 | 파괴 불가능 |
| `H` | 사다리 | 상하 이동 가능 |
| `-` | 바 | 좌우 이동 가능, 매달리기 |
| `$` | 금 | 수집 대상 |
| `T` | 함정 | 벽돌처럼 보이지만 통과됨 |
| `P` | 플레이어 | 시작 위치 (1개만) |
| `E` | 적 | 적 시작 위치 (여러 개 가능) |

## 레벨 설계 규칙

1. 플레이어(P)는 반드시 1개만 배치
2. 금($)은 최소 1개 이상 배치
3. 모든 금에 도달할 수 있는 경로 필요
4. 사다리로 상단까지 탈출 경로 필요
5. 적당한 난이도의 적(E) 배치

```
......#####....H.####.......
...H........E..H............
...H..####.....H.....$......
..$H.......H...H....###.....
###H###....H...H............
.......H...H...H.E....$....
..P....H...H####H...####....
@@@@@@@@@@@@@@@@@@@@@@@@@@@@
```

사용자가 원하는 테마나 난이도를 지정하면 그에 맞는 레벨을 생성하세요.
