# Performance Optimization

game.js의 성능을 분석하고 최적화 방안을 제시합니다.

## 분석 영역

### 1. 렌더링 성능
- Canvas 2D 렌더링 효율성
- 불필요한 재렌더링 여부
- 오프스크린 캔버스 활용 가능성
- requestAnimationFrame 사용 최적화

### 2. 게임 루프
- 프레임 타이밍 일관성
- 업데이트/렌더 분리
- 델타 타임 처리

### 3. 메모리 관리
- 객체 생성/소멸 빈도
- 가비지 컬렉션 영향
- 배열/객체 재사용

### 4. 알고리즘 효율성
- BFS 경로 탐색 최적화
- 충돌 감지 최적화
- 적 AI 계산 효율

### 5. 벡터 그래픽
- Gradient 캐싱
- Path 재사용
- 복잡한 도형 단순화
