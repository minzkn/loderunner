# Lode Runner RULES Regression Checklist (Apple II 1983 Baseline)

기준 문서: `RULES.md`
테스트 빌드: 브라우저에서 `index.html` 실행

## 1. 부팅/기본 상태

1. 타이틀 진입 시 기본 프로파일이 `APPLE2 STRICT`로 표시된다.
2. 시작 목숨이 `5`다.
3. `ENTER`로 게임 시작, `ESC`로 일시정지/복귀가 가능하다.
4. `R` 재시작 시 목숨이 1 감소한다.

## 2. 조작 (Apple II 우선)

1. `I/J/K/L` 이동이 동작한다.
2. `U/O` 파기가 동작한다.
3. `F1/F2` 파기 대체 입력이 동작한다.
4. `M` 사운드 토글이 동작한다.

## 3. 핵심 규칙

1. 점프가 불가능하다.
2. 낙하 데미지가 없다.
3. 플레이어 금 보유량 최대가 1이다.
4. 금(`$`) 획득 시 점수가 `+250` 증가한다.
5. 모든 금 수집 후 탈출 사다리가 활성화된다.
6. 상단 도달 시 레벨 클리어(`+1500`, 목숨 `+1`)가 적용된다.

## 4. 파기/구멍 규칙

1. `BRICK(#)`만 파기 가능하고 `TRAP(T)`는 파기되지 않는다.
2. 목표 상단이 사다리/바이면 파기 불가다.
3. 구멍은 `HOLE_FILL_TIME` 뒤 복구된다.
4. 플레이어는 구멍에서 기본적으로 탈출하지 못한다.
5. 적은 구멍에 갇힌 뒤 탈출 시도를 한다.
6. 구멍 복구 시 적은 사망(+75) 후 리스폰한다.
7. 구멍 복구 시 플레이어가 구멍 안이면 즉사한다.

## 5. false floor / 충돌

1. 플레이어는 `TRAP`에서 1프레임 지연 후 낙하한다.
2. 적은 `TRAP`을 즉시 관통한다.
3. 적과 근접 중첩 시 플레이어가 사망한다.
4. 갇힌 적 위는 일시적 발판처럼 동작한다.

## 6. 엄격 모드 제한

1. 무입력 30초 후 데모가 시작되지 않는다.
2. `E`로 에디터 진입이 차단된다.
3. 자동 STUCK 오버레이가 뜨지 않는다.
4. 탈출 사다리는 숨김 seed(`e`) 공개 방식이 우선 적용된다.

## 7. 프로파일 고정 확인(선택)

1. 타이틀에서 `V` 입력 시 프로파일이 바뀌지 않고 고정 안내 메시지가 표시된다.
2. 모든 플레이가 `APPLE2 STRICT` 기준으로 동작한다.
