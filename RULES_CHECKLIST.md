# Lode Runner RULES Regression Checklist

기준 문서: `RULES.md`  
테스트 빌드: 브라우저에서 `index.html` 실행

## 1. 공통 스모크

1. 타이틀 진입 후 `ENTER`로 시작된다.
2. 플레이 중 `ESC`로 일시정지/해제가 된다.
3. `R`로 현재 레벨을 재시작하며 목숨이 1 감소한다.
4. 금(`$`) 획득 시 점수가 `+250` 증가한다.
5. 모든 금 수집 후 탈출 사다리 활성화 메시지가 표시된다.

## 2. 프로파일 전환

1. 타이틀에서 `V`를 눌러 `APPLE2 STRICT -> C64 STRICT -> MODERN` 순환되는지 확인한다.
2. 타이틀 하단 안내 문구/플레이 중 우상단 배지에 현재 프로파일이 반영되는지 확인한다.

## 3. apple2_strict

1. 시작 목숨이 `5`인지 확인한다.
2. 무입력 30초 후 데모가 시작되지 않는지 확인한다.
3. `E` 키로 에디터 진입이 차단되는지 확인한다.
4. `TRAP(T)` 타일은 파기되지 않고 `BRICK(#)`만 파기되는지 확인한다.
5. 금을 모두 수집하면 숨김 탈출 사다리(seed `e`)가 노출되는지 확인한다.
6. 자동 교착(STUCK) 오버레이가 자동으로 뜨지 않는지 확인한다.

## 4. c64_strict

1. 시작 목숨이 `5`인지 확인한다.
2. 이동 키로 `I/J/K/L`, 파기 키로 `U/O`가 동작하는지 확인한다.
3. `Ctrl+A`로 현재 시도 중단(목숨 1 소모) 동작이 되는지 확인한다.
4. 데모/에디터/자동 교착이 비활성인지 확인한다.
5. 탈출 사다리가 strict 방식(숨김 seed 노출)으로 활성화되는지 확인한다.

## 5. modern

1. 시작 목숨이 `99`인지 확인한다.
2. 이동 키(화살표/WASD/IJKL) 및 파기 키(Z/X/U/O/F1/F2)가 모두 동작하는지 확인한다.
3. 무입력 30초 후 데모 모드가 시작되는지 확인한다.
4. `E` 키로 에디터 진입/종료가 되는지 확인한다.
5. `TRAP(T)` 파기가 허용되는지 확인한다.
6. 교착 상황에서 자동 STUCK 감지가 동작하는지 확인한다.
7. 클래식 150레벨 루프 시 속도 증가 규칙이 적용되는지 확인한다.

## 6. 메카닉 상세 회귀

1. 플레이어 점프가 불가능한지 확인한다.
2. 낙하 데미지가 없는지 확인한다.
3. 플레이어 금 보유량이 최대 1인지 확인한다.
4. 적이 구멍에 빠지면 일정 시간 후 탈출 시도를 하는지 확인한다.
5. 구멍 복구 시 구멍 안 적은 사망(+75) 후 리스폰하는지 확인한다.
6. 구멍 복구 시 플레이어가 구멍 안이면 즉사하는지 확인한다.
7. 탈출 사다리 활성화 후 최상단 도달 시 레벨 클리어(+1500, 목숨+1)되는지 확인한다.
